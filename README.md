#  Langkah langkah 

Berikut adalah langkah-langkah yang dilakukan dalam proses praktikum atau pengerjaan tugas pada repository ini.  
Setiap langkah disertai dengan tampilan (screenshot) dan penjelasannya.
<br><br>
1. Tampilan Awal Program
Pada tahap pertama, program atau proyek dijalankan dan menampilkan tampilan awal seperti berikut.  
Biasanya pada tahap ini kita memastikan semua file sudah terhubung dan tidak ada error pada bagian awal kode.

2. Menampilkan Hasil Eksekusi Kedua 
Langkah kedua menunjukkan hasil proses berikutnya, misalnya ketika kode sudah mulai memproses data atau menampilkan hasil tertentu di terminal. Bagian ini penting untuk memastikan logika program berjalan sesuai urutan yang diinginkan.
